Der CAD-Model-Viewer erlaubt das Einbinden von FBX-Dateien in WordPress Seiten.
Dazu wird WordPress um einen Shortcode erweitert, der über das Backend in Posts und Seiten platziert werden kann.
Das Handbuch enthält eine genauere Beschreibung, sowie eine Instalations- und Gebrauchsanleitung.
